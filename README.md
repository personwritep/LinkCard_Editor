「LinkCard Editor」は、アメーバブログ専用のリンクカードのアレンジ編集ツールです。<br>
このツールはリンクカードの細部構成の加工・編集が可能で、編集結果はブログ記事上のリンクカードが要求される機能・デザインを充分にカバーします。 このツールはアメーバのブログ編集画面でのみ実行可能ですが、そこで生成・整形したリンクカードは、多くの場合に HTMLコピーで他のブログ等でも利用できます。<br>
<br>
